#### 极简版（我个人使用）

使用 `typora.app`，然后使用 `vue` 白色或者黑色主题（建议白色），打印出 pdf 即可，如果需要添加头像，则需要导出为 html 文件，然后自定义 img 头像样式和定位。

#### 备注
1. 简历一定要用 pdf，这样整体简历样式比较好，而且不会乱码。看到很多简历喜欢用 word，其实可以转换为 pdf，word 打开麻烦 && 样式一般。
